Clients

{% include technology.html %}

Buddycloud clients are for users to access the Buddycloud social network. A client can connect to any server and use any JID. Users can befriend any other user on a public Buddycloud server (servers are federated).

Web Client

The web client is an html5 javascript application for end users to connect with their friends. It implements a real-time social network. We are trying to create something that feels familiar to Facebook users but utilises the unique features of Buddycloud. It is written in Coffeescript, backbone.js and jQuery. The web client uses aggresive client side caching to get sub-second load times.

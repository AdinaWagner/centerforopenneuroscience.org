You can access Buddycloud in many ways. The following steps quickly introduce you on how to exchange posts in a channel node, which is the most basic thing you can do with Buddycloud.

This example uses Javascript but you can use our [RESTful API](/api) in order to build other apps such as mobile ones.

Register an account

The following code snippet issues a POST request to our REST API in order to create an account. 

      
        $.ajax({
            type: "POST",
            url: "demo.buddycloud.org/api/account",
            contentType: "application/json",
            processData: true,
            data: JSON.stringify({
                username: "",
                password: "my-secret-password",
                email: "my-email@email.com"
            }),
            success: function(data){}
        });

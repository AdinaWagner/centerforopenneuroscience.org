Full Stack

Connect your users with each other through chat, status updates, file sharing and micro-blogs.

Buddycloud does the heavy lifting for you: with a protocol, servers for push, media and file sharing, clients, libraries and API you have all the bits you need.

Own it

You decide the server - run your own or use the hosted option. Hosted gives you quick start. Run your own gives you complete control and privacy.

You decide which bits to run: add push notifications, add media sharing, add chat. Buddycloud is an ecosystem of components that interconnect to save you time developing them.

You decide to run Buddycloud separate from other Buddycloud sites or kick start your application growth by interconnecting with other sites and existing Buddycloud users.

Privacy 2.0

Build on a platform with strong privacy and security at its core.

For Organisations
Keep your and your team's information secure by running your own Buddycloud server inside your organisation's network. You control which information you share outside the network. Your organisation's communication data stays private.
For Developers
Protect users' privacy with Buddycloud's secure design and inspectable open source code and protocol.
For Users
Stay private with simple sharing controls.

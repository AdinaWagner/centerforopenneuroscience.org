Here's a selection of Buddycloud services and modules to get you started.

Name Description Source Package Getting started Contribute
Buddycloud server stores, synchronises and updates Buddycloud chanels.
API server RESTful access to all Buddycloud functions
push server Configure and push Buddycloud events to mobile and email
Buddycloud for Android Access Buddycloud channels on Android
Buddycloud webapp Backbone based Buddycloud app
install tester Test your own Buddycloud install
Buddycloud media server Share and synchronise media (and avatars between different sites running Buddycloud)

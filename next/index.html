buddycloud features

Celebrating protocol: open protocols, open source, open ecosystem

buddycloud and the W3C, Mozilla Foundation, and XSF are working together to buile the strong foundation for an open social ecosystem.

Using open standards means that you don't have to depend on anyone else and can quickly finish your project using a wide range of free libraries.

buddycloud is built by and for thinkers and makers who care about openness.

All buddycloud software is licensed under the liberal Apache 2.0 license: reference software for a webclient and Android gets you running fast.

fork the buddycloud software and kickstart your project

  • Avoid re-implementing a social stack on every single new project: buddycloud ships with working, Apache 2.0 licensed code, that will help get you started quickly.
  • The server is written in node.js, the webclient is a single-page backbone.js app, and and we have developed a native Android client.
  • All code is designed so that you can quickly and easily adapt and license under a do-what-you-want-with-it license.

Distributed by design

How would it be if Gmail couldn't send email to a Hotmail account? Today social networks are designed like that.

buddycloud has designed a new way to share: each buddycloud instance can connect and share information with other buddycloud websites. Or run it behind your firewall for just your team.

Also, we have designed tools that tap into Facebook and Twitter to help connect with users on "legacy" social networks.

When you start your own buddycloud, your small website joins a much larger ecosystem of other domains also running buddycloud and users don't need to register again and again.

privacy

  • SSL client to server encryption
  • SSL communication between servers encrypted
  • Users create private or open spaces (channels) to share content.
  • You control how data on your buddycloud is shared with the wider world

media sharing

Media can be shared within just one channel or with the entire world.

The buddycloud media server is like DropBox enhanced to work on every domain: now users and developers can manage their own media sharing. Files, films, photos are seamlessly synchronised and it just works. Learn more.

run your own buddycloud or have it hosted

We want you to start developing quickly and then be able to run your own server infrastructure.

You could install your own buddycloud stack: API, media, pusher, and buddycloud servers.

Or, to get going faster we can host your social API and, when you grow, you can migrate out your data, users and media onto your own server.

A helpful community

We know that changing the world is difficult. If you get stuck, we're a pretty helpful and enthusiastic bunch and are here to help you.

Ask for help on the dev list, the chat room or in the buddycloud team channel.

Commercial licensing and support

Sometimes spinning up a large buddycloud deployment needs a little extra help. We want to see buddycloud deployed everywhere and we've got you covered.

We provide ongoing support to corporations, telcos, and software houses to to help them keep their buddycloud servers running smoothly.

buddycloud also provides consulting. For example, scaling a telco's buddycloud software stack to the multi-million simultaneous-user level.

recommendation engine

If you like the buddycloud channel food@foo.com then might be interenestd in omnomnom@bar.com. buddycloud includes a crawler with a nice API that helps your users find great content.

realtime search engine

Find channels about cooking or posts about siamese cats. Fast. Public content on buddycloud is quickly indexed and searchable and accessible via a standardised API and firehose feed.

Social discovery engine

The buddycloud friend finder helps your users find their friends on other social networks. The FF uses hashes of identifiers: for example Twitter or Facebook account IDs. This extends the usefulness to existing circles of friends.
